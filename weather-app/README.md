# 🌦️ Weather Widget React App

Une petite application React qui permet d'afficher la météo d'une ville choisie, avec prise en charge multilingue (FR/EN) et adaptation du fond selon la météo.

---

## 🚀 Lancer le projet en local

### 1. Cloner le dépôt
```bash
git clone https://github.com/Nicolas-Bourahoui/WeatherWidget.git
cd WeatherWidget
```

### 2. Installer les dépendances
Si Node.js est déjà installé
```bash
npm install
```

Sinon installer Node.js ici : https://nodejs.org/en puis faites les commandes précédente


### 3. Lancer l'application
Si Node.js est déjà installé
```bash
npm start
```
L’application va démarrer automatiquement sur http://localhost:3000 dans ton navigateur par défaut.


---

🔧 Fonctionnalités
🔍 Recherche de ville via un champ input

🌐 Choix de la langue (Français / Anglais)

🌡️ Affichage de la température, humidité, couverture nuageuse et d'autres...

🎨 Changement automatique du fond selon la météo

📱 Interface responsive


🌤️ API utilisée
OpenWeatherMap API
